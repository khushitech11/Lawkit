⚖️ Lawkit

Lawkit is an Android application designed to simplify access to Indian criminal laws. It enables users to search IPC laws, understand legal provisions, and access lawyer information through an easy-to-use interface.

 Features

- 🔍 Search IPC laws
- 📖 View detailed legal information
- 👨‍⚖️ Browse lawyer details
- 🔐 User authentication
- ☁️ Firebase integration
- 📱 Simple Android interface

## Technologies Used

- Java
- Android Studio
- Firebase Realtime Database
- HTML
- CSS
- JavaScript

Files Included

- Lawkit.apk – Android application
